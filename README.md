# everest
Test exercise for ITLeague

## Project setup
```
yarn
```

### Compile and enable hot-reloader for development
```
yarn run serve
```

### Build bundle for production
```
yarn run build
```

### Run tests
```
yarn run test
```

### Lint and fix files
```
yarn run lint
```

### Deploy bundle on Now service
```
npm i -g now
yarn run deploy
```
